About me

I am a passionate Full Stack Web Developer with expertise in both front-end and back-end development. My main focus is working with the MERN stack (MongoDB, Express, React, and Node.js), which allows me to build efficient, scalable, and dynamic web applications.

With a strong foundation in front-end technologies like React, I enjoy creating interactive and intuitive user interfaces that provide great user experiences. On the back-end, I leverage Node.js and Express to build robust APIs and manage data with MongoDB, ensuring seamless integration and performance. I’m always excited to work on innovative projects and continually enhance my skills by staying updated with the latest industry trends.
